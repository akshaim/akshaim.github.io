Setting up Esplorer

We will be using Esplorer , an IDE for ESP8266, for programming the chip.The software is written in Java and is compatible with machines running Java SE or later

  1. Install Java.
    sudo add-apt-repository ppa:webupd8team/java -y
sudo apt-get update
sudo apt-get install oracle-java8-installer
  2. Download Esplorer from here and extract it.
  3. Launch Esplorer 
    sudo java -jar ESPlorer.jar
