Debugging

Introduction to Monitoring and Debugging using 4DIAC.

4DIAC offers Debugging Perspective to monitor and debug applications.The perspective offers following options

  1. Watch
  2. Trigger
  3. Force

Watch

Watch enables a user to view the status of event(s) and data

  • Enable Monitoring
  • Enable Watch
    • The video shows the watch of application deployed in Chapter 2. Make sure that the PC running 4DIAC and EMB_RES (RPi) running Forte is connected to a network to view the same.
